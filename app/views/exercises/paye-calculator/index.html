Exercise: reproducing the PAYE calculator

Please reproduce the PAYE calculator. This page isn't in the GOV.UK style but please use components and styles from GOV.UK Design to create this page.

Try to upload your page and attach your link to the last section of this page. Remember to create a branch off the main. When you are ready to publish it, you can create a pull request and ask another designer to do the code review for you.

General rules

  • Re-produce the calculator.
  • Add the results page.
  • Use styles and components from GOV.UK Design System.
  • No need to reproduce the header and footer.
  • No need to produce a realistic calculator. The calculation result is not important.
  • No need to make it exactly the same.

Techniques you may apply

  • Use of Layout from Design System
  • Use of Typography from Design System
  • Use of form component from Design System
  • Learn how to produce multi-page prototype
